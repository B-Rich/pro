Box is based on Ubuntu 16.04. According to that you need to use the newest Vagrant (<= v1.8.1). It solves the issue with new naming for network interfaces.

To use Pro Vagrant box make sure there is no node_modules directory in servers/webpack or servers/primus then use following commands: {% highlight bash %} vagrant up vagrant ssh cd /vagrant foreman start {% endhighlight %}
